## 팀 구성 및 역할
✅ 이성우(팀장) : 강아지 추적 알고리즘 구현, 데이터베이스 구축


✅ 김민정 : 강아지 추적 알고리즘 구현, QT 기반 자반기 UI 개발


✅ 김정대 : TCP/IP 소켓 통신 구축, STM 기반 자판기 펌웨어 개발


✅ 정희중 : 자판기 하드웨어 제작


✅ 조수환 : QT 기반 자판기 UI 개발, 견종 분류 모델 개발 및 최적화
